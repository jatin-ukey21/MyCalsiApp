# Java Calculator GUI  

## How to Run  
1. **Install Java** (if not installed).  
2. Double-click `run.bat` (Windows) or run `java -jar Calculator.jar` (if JAR exists).  

## Files Included  
- `CalculatorGUI.java` (Source code)  
- `java-cup-11b-runtime.jar` (Required library)  
- `run.bat` (Launch script)
